## 修复内容

### 搜索与爬取衔接

- 修复知乎搜索结果 CSV 的“链接”列无法被详情爬虫读取的问题。
- 修复抖音搜索结果 CSV 的 `url` 列无法作为后续爬虫输入的问题。
- 修复知乎搜索模块缺少 `Path` 导入、模块加载即失败的问题。

### 数据处理与稳定性

- 让 `batch_labeler.py` 同时支持旧浏览器导出字段与当前知乎爬虫字段。
- 修复知乎专栏文章标题丢失。
- 失败的知乎请求不再被断点续跑误认为已完成；旧的空白失败行也会重试。
- 修复空 CSV、无待爬评论时的崩溃路径。
- 修复旧抖音 `CsvPipeline` 对不存在字段的引用。
- 统一抖音视频时长为秒。

### 抖音批次合并与 CDP

- 合并多轮抖音爬取结果时，同一视频优先保留含评论、字段更完整的记录。
- 排除“合并/清洗后”等中间产物，避免被再次当成原始输入。
- 合并输出使用全量字段并集，避免评论列因首个文件字段较少而丢失。
- 在环境中加入 `websocket-client`；Chrome 调试端口不可用时，CDP 工具安全返回不可用状态。

## 文档与操作规范

更新 `README.md`、完整教程和 `CLAUDE.md`，明确抖音关键词搜索前的必备流程：

1. 在 Windows PowerShell 以 `--remote-debugging-port=9222` 与 `--remote-allow-origins=*` 启动独立 Chrome。
2. 在该浏览器中登录抖音并保持 `www.douyin.com` 页面打开。
3. 在项目终端运行 `get_ws()` 验证命令，确认输出 `READY`。
4. 再执行 `source.douyin_crawler.search_main`。
